apply a one time pad
Usage: otp [OPTIONS] KEY TARGET
KEY
	the key file to XOR with TARGET
OPTIONS
	-0 INT
		seek to INT before writing the output
	-b INT
		set the buffer length manually
		(for security, keep this small; defaults to 512B)
	-c INT
		the count for which to apply
	-h
		print this text and exit
	-k INT
		seek to INT before reading the KEY
	-o PATH
		change the output PATH
		(setting this to an alternate path increases the number of copies
		of the output, so use it sparingly; defaults to TARGET)
	-t INT
		seek to INT before reading the TARGET
TARGET
	the input file to modify,
	and (unless `-o` is specified), the output location
