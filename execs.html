execs 1.0 execute arbitrary instructions
Unix-like

Accepts instructions in hexadecimal (i.e. can execute instructions containing NULs). Performs a targeted buffer overflow attack on itself. Built with the GNU Compiler Collection and the x64-64 architecture in mind. Can be reconfigured (by modifying the definition of RET2P_RA_OFFSET) to work with different compilers/architectures.
