NAME

arachnid – a flexible search program

SYNOPSIS

arachnid [OPTIONS] [PATH] STRING

DESCRIPTION

Arachnid is intended to unify many types of searches one might perform at the terminal. The types of searches themselves fall into the categories of file content searches and and path component searches. A file content search is modifiable via the −b option, which specifies an alternative buffer size. A path search is modifiable via the −p, −n, and −e options. These respectively specify to search full paths, path names, or extensions. If more than one of these options are provided, the largest is used. Their order of precedence is as such: −e < −n < −p. Using these options in addition to the -w option, one can specify paths to be searched. See the examples for more detailed usage.
−b, −−buflen=INT

use a custom buffer size (in bytes)

−c, −−nocolor

don’t colorize output

−C, −−nocount

don’t show match count

−e, −−extonly

search path extensions only

−i, −−ignorecase

ignore case

−h, −−help

display this text and exit

−l, −−followlinks

follow all symbolic links

−n, −−nameonly

search final components of path names only

−p, −−pathonly

search full path names only

−s, −−showspecial

include all special devices

-w, −−with=STRING

if searching extensions, file names, or paths was specified, search paths that contain STRING in the specified component

The PATH argument is a valid path. If omitted, the CWD is used. If ’−’, search STDIN.

The STRING argument is the search string, and is not optional. The search string may be empty.

EXIT STATUS

0

OK

1

Error

EXAMPLES

arachnid string

For every file beneath the CWD, if "string" appears in the file, print the number of occurrences and the corresponding path.

arachnid −i /usr/share/man/man1/man.1 .SH

Print the number of sections in man’s manual page.

arachnid ’’

Prints the file size + 1 and path of every file beneath the CWD.

arachnid −lpCs ’’

Prints every accessible path beneath the CWD.

arachnid −e / .conf

Prints all configuration files beneath the root directory.

arachnid −ew .txt string

For each file beneath the CWD with the ".txt" extension, print the number of occurrences of "string" and the corresponding path.
